# config
This is a collection of my vim, tmux, and other configs.

# thanks

# install
You must have git installed. and..
```bash
cd ~ && git clone https://github.com/labeldock/config.git
```

# setting
```bash
#bash
vim ~/.bash_profile

#paste
[[ -s "$HOME/config/unix-init.sh" ]] && source "$HOME/config/unix-init.sh"

#:wq

# reopen terminal

config
0!
```
